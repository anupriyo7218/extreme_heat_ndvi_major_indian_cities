# 🌍 Extreme Heat & NDVI Analysis in Major Indian Cities

## 📌 Project Overview

This project analyzes rising temperatures and vegetation patterns across major Indian cities:

* Kolkata
* Delhi
* Mumbai
* Bangalore

Using satellite-based datasets (MODIS), the project explores:

* Heatwave intensity (Land Surface Temperature)
* Vegetation health (NDVI)
* Relationship between temperature and vegetation

## 📊 Analysis (Jupyter Notebook)

The notebook includes:

* 📈 Yearly temperature trends (2020–2025)
* 🔥 Extreme heat days (>35°C)
* 🌡️ Heatwave intensity comparison
* 🏙️ Multi-city comparison
* 🗺️ Spatial visualization
